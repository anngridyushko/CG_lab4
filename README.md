# Лабораторная работа 4
**Компьютерная графика**

**Описание:** Приложение, иллюстрирующее работу базовых растровых алгоритмов. Лабораторная написана на языке C#.

**Установка**: Достаточно скачать архив, разархивировать его и запустить исполняемый файл CG_lab4.exe

**Использование**: Для алгоритмов ЦДА, Брезенхема и пошагового необходимо ввести целые числа в поля для координат точек (x1, y1) и (x2, y2). Для растеризации окружности алгоритмом Брезенхема нужны поля для центра окружности (x1, y1) и радиуса R.

Вид главного окна приложения:
![image](https://user-images.githubusercontent.com/61574185/115615446-a859a080-a2f7-11eb-9d70-76e12e4e0ca7.png)

Приложение позволяет регулировать масштаб. Для приближения и отдаления используются ЛКМ и ПКМ соответственно, при этом масштаб изменяется каждый раз в 2 раза. Таким образом приближать можно до 4 раз.
